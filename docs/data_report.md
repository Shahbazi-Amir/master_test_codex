# گزارش وضعیت داده‌های واقعی آزمون برق ۱۲۵۱

هدف پروژه، شبیه‌سازی واقعی آزمون کارشناسی ارشد مهندسی برق کد ۱۲۵۱ است. بنابراین داده نمونه یا ساختگی نباید وارد آزمون اصلی شود.

## سیاست فعلی

- سؤال‌های نمونه از UI اصلی حذف شده‌اند.
- هر سال فقط وقتی فعال می‌شود که سؤال، گزینه‌ها و کلید از منبع واقعی استخراج و کنترل شده باشد.
- وضعیت لازم برای نمایش سؤال در UI: `extracted_verified`.
- رتبه و قبولی تا زمان ورود داده معتبر فعال نمی‌شود.
