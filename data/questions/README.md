# داده‌های سؤال‌ها

این پوشه فقط برای سؤال‌های واقعی آزمون کارشناسی ارشد مهندسی برق کد ۱۲۵۱ است.

قانون پروژه:

- سؤال نمونه، ساختگی یا آموزشی در آزمون اصلی ممنوع است.
- هر سؤال باید متن سؤال، چهار گزینه، کلید معتبر و منبع داشته باشد.
- فقط رکوردهایی که `extraction_status` آن‌ها برابر `extracted_verified` باشد در UI آزمون نمایش داده می‌شوند.
- اگر PDF پیدا شده ولی استخراج متن قابل اعتماد نیست، فایل سؤال JSON ساخته نمی‌شود.

ساختار رکورد معتبر:

```json
{
  "year": 1404,
  "exam_code": "1251",
  "lesson": "نام درس",
  "question_number": 1,
  "question_text": "متن سؤال واقعی",
  "options": ["گزینه ۱", "گزینه ۲", "گزینه ۳", "گزینه ۴"],
  "correct_option": 1,
  "explanation": "اختیاری؛ فقط با منبع معتبر یا تولیدشده با برچسب جداگانه",
  "source_url": "لینک منبع",
  "source_type": "official_pdf یا trusted_pdf",
  "extraction_status": "extracted_verified"
}
```
