# داده‌های قبولی و رتبه

تا وقتی داده واقعی و قابل استناد برای رتبه، ظرفیت، گرایش و محل قبولی وارد نشده است، تخمین رتبه و قبولی در UI اصلی فعال نمی‌شود.

قانون پروژه:

- عدد ساختگی برای رتبه یا قبولی ممنوع است.
- هر داده باید سال، گرایش، دانشگاه، دوره، ظرفیت/رتبه و منبع داشته باشد.
- خروجی قبولی فقط وقتی فعال می‌شود که داده ورودی کافی و منبع‌دار باشد.
