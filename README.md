# بانک آزمون‌های کارشناسی ارشد

این مخزن داده و اپ اولیهٔ آزمون‌های کارشناسی ارشد را با حفظ تصویر کامل سؤال‌ها، فرمول‌ها، نمودارها و گزینه‌ها نگه‌داری می‌کند.

## آزمون‌های موجود

| رشته | کد مجموعه | سال | تعداد سؤال | وضعیت کلید |
|---|---:|---:|---:|---|
| مهندسی برق | 1251 | 1404 | 125 | رسمی |
| مهندسی کامپیوتر | 1277 | 1404 | 115 | رسمی |
| مهندسی کامپیوتر | 1277 | 1403 | 115 | نهایی؛ سؤال‌های 46 و 114 حذف شده‌اند |
| مهندسی کامپیوتر | 1277 | 1402 | 115 | کلید اولیه رسمی |

برای هر آزمون کامپیوتر، PDF سؤال‌ها، PDF کلید، ۱۱۵ تصویر سؤال، JSON کامل آزمون و JSON مستقل کلید ذخیره شده است. تصویر هر سؤال صفحهٔ کامل دفترچه است تا متن مشترک یا اجزای تصویری ناقص بریده نشوند.

```text
assets/
├── source/computer_engineering/{1402,1403,1404}/{questions,answer_key}.pdf
└── questions/computer_engineering/{1402,1403,1404}/question_001.png ... question_115.png
data/
├── answer_keys/computer_engineering/key_{1402,1403,1404}.json
└── questions/computer_engineering/exam_{1402,1403,1404}.json
```

## اجرا

```bash
pip install -r requirements.txt
streamlit run master_exam_ui.py
```

اپ امکان انتخاب آزمون و درس، پاسخ‌دادن با چهار گزینه، جابه‌جایی بین سؤال‌ها، دانلود PDFها و محاسبه نتیجه با نمره منفی را دارد. سؤال‌های حذف‌شده وارد محاسبه نمی‌شوند.

## پاسخ‌های تشریحی

ساختار `explanation`، `explanation_status` و `explanation_sources` در JSON آماده است. پاسخ تشریحی تأییدنشده منتشر نشده و موارد فعلی با وضعیت `pending_review` مشخص شده‌اند.

## بازسازی داده‌های مهندسی کامپیوتر

اسکریپت `tools/build_computer_exams.py` با دریافت پوشه‌ای شامل PDFهای مبدأ و تصاویر رندرشده، خروجی‌های نسخه‌بندی‌شدهٔ هر سه سال را دوباره تولید می‌کند.
