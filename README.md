# آزمون‌های ارشد مهندسی برق

سامانه آزمون کارشناسی ارشد مهندسی برق، کد مجموعه `1251`، برای سال‌های ۱۴۰۲، ۱۴۰۳ و ۱۴۰۴.

## داده‌های موجود

| سال | کد دفترچه | تعداد سؤال | PDF و کلید | تصاویر سؤال |
|---|---:|---:|---|---:|
| ۱۴۰۲ | `835C` | ۱۳۳ | موجود | ۱۳۳ |
| ۱۴۰۳ | `170C` | ۱۳۱ | موجود | ۱۳۱ |
| ۱۴۰۴ | `535C` | ۱۲۵ | موجود | ۱۲۵ |

هر رکورد سؤال شامل درس، شماره، چهار گزینه انتخابی، پاسخ صحیح و مسیر تصویر است. هر تصویر، صفحه کامل رسمی مربوط به سؤال را نگه می‌دارد تا فرمول، نمودار، متن مشترک و گزینه‌ها ناقص بریده نشوند.

```text
assets/
├── source/arshad_bargh_{year}_1251.pdf
└── questions/{year}/question_001.png ...
data/
├── answer_keys/key_{year}.json
└── questions/exam_{year}.json
master_exam_ui.py
```

## اجرا

```bash
pip install -r requirements.txt
streamlit run master_exam_ui.py
```

در نوار کناری اپ می‌توان سال، درس و شماره سؤال را انتخاب کرد. پاسخ‌ها برای هر سال جدا نگه‌داری می‌شوند.

## پاسخ‌های تشریحی

فیلدهای `explanation`، `explanation_status` و `explanation_sources` در JSON تعریف شده‌اند. در نسخه فعلی فقط کلید اولیه سنجش ثبت شده و پاسخ تشریحی تأییدنشده منتشر نشده است.

## بازسازی داده‌ها

اسکریپت‌های `tools/build_exam_1404.py` و `tools/build_exam_years.py` ساخت داده‌ها را مستند می‌کنند. اسکریپت سال‌های ۱۴۰۲ و ۱۴۰۳ انتظار دارد PDF و صفحات رندرشده ورودی در مسیر موقت مشخص‌شده داخل فایل موجود باشند.
